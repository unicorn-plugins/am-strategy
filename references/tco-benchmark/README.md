## 데이터 한계와 확보 권장 활동

### 본 디렉토리의 데이터 한계 (정직한 명시)

1. **공공부문 외 업종(금융·제조·유통)별 정밀 TCO 벤치마크는 2025년 이후 공개 자료 부재**
   - 금융권: 클라우드 *도입 동향* 자료는 풍부하나 *TCO 절감률* 공개 데이터 없음
   - 제조·유통: 산업 평균 정량 데이터 미공개 (개별 기업 사례 출처 검증 어려움)
2. **사례성 수치 (예: "60% 절감")는 2024년 이전 출처 또는 출처 불명** → 본 디렉토리에서 제외
3. **공공부문 18.4%는 *7년 누적*** — 단년 효과로 환산 시 주의

### 보충 데이터 확보 권장 활동

| 활동 | 방법 |
|------|------|
| 동종업계 벤치마크 | 대상 기업이 직접 동종업계 3~5개사와 비공식 데이터 교환 |
| 가트너 한국 컨설팅 | 유료 (2,000~5,000만원/건) — 업종별 TCO 정밀 데이터 확보 가능 |
| 한국IDC 맞춤 보고서 | 유료 — 업종별 클라우드 채택률·CAGR 정밀 분석 |
| 글로벌 보충 | DORA 2025 5메트릭 ([`../dora/`](../dora/)) — 업종 무관 글로벌 표준 |
